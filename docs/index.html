💰 SpendWise

A personal expense tracker built with the LAMP stack, containerized with Docker and refactored into MVC architecture.

View on GitHub MVC Branch

Tech Stack

PHP 8.2 Apache MySQL 8.0 Docker Bootstrap 5 MVC Architecture PDO Prepared Statements bcrypt Auth Stored Procedures

Features

🔐
Secure Authentication

User registration and login with bcrypt password hashing and PDO prepared statements to prevent SQL injection.

📊
Monthly Summary

MySQL stored procedure aggregates expense totals by category per month with month-by-month navigation.

🏗️
MVC Architecture

Refactored from procedural PHP into clean MVC structure separating Models, Views, and Controllers on a dedicated Git branch.

🐳
Dockerized

Fully containerized with Docker Compose — runs with a single command on any machine with Docker installed.

💳
Full CRUD

Add, edit, and delete expenses with category tagging, date tracking, and real-time total calculations.

📱
Responsive UI

Clean, responsive interface built with Bootstrap 5 — works on desktop, tablet, and mobile.

Getting Started

Requires Docker Desktop installed and running. 

git clone https://github.com/Helenyixuanwang/spendwise.git
cd spendwise
docker compose up -d

Then visit http://localhost:8080/auth/register.php to create an account.
